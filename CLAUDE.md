# VeganGlow

## Overview
E-commerce platform for vegan cosmetics. Next.js 16 + Supabase + Capacitor.js monorepo.

## Key Commands
- `npm run dev` — Start frontend dev server
- `npm run build` — Production build
- `npm run lint` — ESLint check
- `npm run type-check` — TypeScript validation

## Important Files
- `frontend/src/lib/supabase/` — Supabase client setup
- `frontend/src/types/database.ts` — Generated DB types
- `backend/supabase/migrations/` — Database schema
- `backend/supabase/functions/` — Edge Functions (Deno)
